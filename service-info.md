# WAV → MP3 변환기 서비스 정보

## 서비스 기본 정보

- **서비스명**: WAV → MP3 변환기
- **URL**: https://wav.funnyfunny.cloud/
- **가이드 URL**: https://wav.funnyfunny.cloud/guide.html
- **카테고리**: 🎵 오디오 편집 / 🧰 유틸리티
- **태그**: `audio`, `converter`, `mp3`, `wav`, `local`, `browser`, `ffmpeg`, `utility`

## 서비스 설명

브라우저에서 WAV 파일을 MP3로 변환하는 도구입니다. 서버 업로드 없이 100% 로컬에서 처리되어 개인정보와 파일이 안전하게 보호됩니다.

### 주요 특징

- ✨ 서버 업로드 없음 - 모든 처리가 브라우저에서 이루어짐
- 🔧 ffmpeg.wasm 기반 - 강력한 오디오 인코딩 엔진
- 🎚️ 다양한 비트레이트 선택 - 96kbps부터 320kbps까지
- 🆓 무료 사용 - 로그인 없이 바로 사용 가능

## 사용 방법 요약

1. 변환 엔진 로드 버튼 클릭
2. WAV 파일 선택
3. 비트레이트 선택 (선택사항, 기본 128kbps)
4. MP3로 변환 버튼 클릭
5. 다운로드 링크에서 파일 다운로드

## 주의사항

- 파일이 크면(수십~수백 MB) 변환에 시간이 걸립니다
- 모바일은 메모리 부족으로 실패할 수 있습니다
- 권장: 100MB 이하의 파일 사용

## 기술 스택

- **라이브러리**: @ffmpeg/ffmpeg (ffmpeg.wasm)
- **처리 방식**: 클라이언트 사이드 (브라우저)
- **지원 형식**: WAV → MP3
